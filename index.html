Fun Coding Projects

When I started programming everything was on the command line and felt a lot more straight forward. This is a collection of command line text and numerousy games that all use the Python standard library (the tools included by default when you download Python). Initial projects just use text, numbers, selection and iteration. Subsequent projects will use the curses library for creating more advanced command line apps. Eventually I hope to add some simple pygame projects (not in the standard library.
